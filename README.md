![Preview](preview.png)

This repo contains my personal dotfiles, which I use for study and reading.

## Programs
- Distro: Fedora 33
- WM: spectrwm
- Editor: nvim
- Terminal: urxvt
- Launcher: dmenu and rofi
- Music Player: cmus
- Audio Visualizer: glava
- Notifications: dunst
